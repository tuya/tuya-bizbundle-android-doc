1. 框架接入

【注意事项】

我们目前已经废弃 maven { url 'https://maven-other.tuya.com/repository/maven-public/' } 请使用 jcenter() 源

使用 SNAPSHOT 版本的组件请去除 SNAPSHOT 后缀

1.1. 业务包配置文件

1.1.1. assets 配置

module_app.json 为服务化配置文件，拷贝 module_app.json 至 app 目录下的 assets 文件夹下即可生效。[必选]

x_platform_config.json 为商场和设备控制业务包服务化配置文件，在接入商场和设备控制业务包时拷贝 x_platform_config.json 至 app 目录下的 assets 文件夹下即可生效。[可选]

1.1.2. res 配置

res/values多语言资源文件，可根据情况选择配置 [可选]

1.1.3. 依赖源配置

根目录的 build.gradle 文件

allprojects {
    repositories {
        jcenter()
    }
}

app 的 build.gradle 文件

android {
    defaultConfig {
        ndk {
            abiFilters "armeabi-v7a", "arm64-v8a"
    }
}
